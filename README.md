
  # Invoice Fraud Detection App

  This is a code bundle for Invoice Fraud Detection App. The original project is available at https://www.figma.com/design/spVA6OoEBJVQr3L3Q0umEs/Invoice-Fraud-Detection-App.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  Deployment Note: The system runs locally with the provided setup instructions. Due to hackathon time constraints, full production deployment was not finalized.
